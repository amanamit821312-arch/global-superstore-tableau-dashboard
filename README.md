# global-superstore-tableau-dashboard
Interactive Tableau dashboard analyzing Global Superstore sales data with insights into revenue, profit, customer segments, regional performance, and business trends through dynamic visualizations.
## Project Overview

This project presents an interactive Tableau dashboard built using the Global Superstore dataset. The dashboard provides valuable insights into sales performance, profitability, customer behavior, and regional trends through dynamic visualizations.

## Objectives

* Analyze overall sales and profit performance.
* Identify top-performing products and categories.
* Evaluate regional and market-wise sales trends.
* Monitor customer segment performance.
* Discover business opportunities through data-driven insights.

## Tools Used

* Tableau
* Microsoft Excel / CSV Dataset

## Dashboard Features

* Sales Performance Analysis
* Profit Analysis
* Regional Sales Overview
* Customer Segment Insights
* Product Category Performance
* Interactive Filters and Drill-Down Analysis

## Key Insights

* Comparison of sales and profit across regions.
* Identification of top-selling products and categories.
* Customer segment contribution to revenue.
* Market-wise business performance trends.
* Analysis of profitable and loss-making areas.

## Dataset

The dashboard uses the Global Superstore dataset containing order details, customer information, product categories, sales, profit, and shipping data.
